### How to Run the App

- Open the terminal and run **npm install** inside the project root directory.
- Create a **.env** file with the following:

**VITE_EDAMAM_APP_ID= your API ID**<br/>
**VITE_EDAMAM_APP_KEY= your API KEY**<br/>

- Execute **npm run dev** and click on the link to open the app in the browser.

## How to Use the App

1. Enter the recipe keyword in the keyword inputfield e.g "Chicken"
2. Enter an ingredient in the ingredient inputfield e.g "pepper"
3. Click the add button and the ingredient inputfield will be cleared.
4. Add another ingredient and click the add button, if you want.
4. Click the search button and the recipe will be displayed.
